## Получение сборка и проверка

## Получение и сборка на Linux
    git clone https://github.com/ud84/wui
    cd wui
    cmake CMakeLists.txt
    make

## Проверка на Linux
    cd examples/simple
    ./simple

## Получение и сборка на Windows
    git clone https://github.com/ud84/wui
  
Откройте wui.sln при помощи Visual Studio и соберите проект

## Проверка на Windows
Запустите 
    simple.exe
